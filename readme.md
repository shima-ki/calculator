


## 動作環境

- Laravel 5.4
- vue.js 2.0.8
- SQLite3

## 必要なソフト
- Composer
- Node.js


## 動作手順

1.Calculatorをダウンロードする
2.コンソールを起動し、ダウンロードしたフォルダ（Calculator-master）に移動する
3.以下のコマンドを実行する
- composer install
- npm install
- npm run dev
