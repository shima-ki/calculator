## 構築環境

- Laravel 5.4
- vue.js 2.0.8
- SQLite3

## 必要なソフト
- Composer(https://getcomposer.org/)

## 動作手順

1.任意のフォルダにgit cloneを実施
	- git clone https://github.com/shima-ki/calculator.git
2.コンソールを起動し、cloneしたフォルダに移動する  
3.以下のコマンドを実行する
	- composer install
	- php artisan serve
4.http://localhost:8000にアクセスをする
